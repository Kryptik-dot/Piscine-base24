# Calculatrice en Base 24

## Informations Générales
- **Auteurs** : BOUZANGA Ruben & PAMBOU Emmanuelle & EMBAMBA ELI 
- **Classe** : BFSDA1
- **Base numérique** : 24 (0-9, A-N)

## Fonctionnalités
1. **Lexeur structuré** : Analyse la chaîne de caractères et identifie les tokens (Nombres, Opérateurs, Parenthèses).
2. **Gestion des priorités** : Implémentation de l'algorithme Shunting-yard pour respecter les priorités mathématiques (* et / avant + et -).
3. **Tests Unitaires** : Le programme inclut une suite de tests automatiques à l'initialisation pour garantir la fiabilité des calculs.
4. **Gestion de fichiers** : (Optionnel) Exportation des résultats.

## Comment l'utiliser
Saisissez simplement une expression en base 24. 
Exemple : `(A + 5) * 2`
